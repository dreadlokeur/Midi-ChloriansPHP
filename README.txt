SYSTEM REQUIREMENTS
-------------------
MidichloriansPHP requires PHP 5.4 or later. 

FEATURES
--------
 * MVC architecture.
 * Auto-loading class and configuration.
 * Multi-language.
 * Asynchronous request (AJAX).
 //* Pipelining HTTP.
 * Configuration multi format (XML, INI, YAML).
 * Templating multi format (PHTML, TWIG, SMARTY) and multiple.
 * Security components.
 * Profiling and debug tools.
 * Logging system.
 * Errors management.
 * Caching engine.
 * Network utility.
 * Database access layer (DAL).
 * Routing and url management.
 * Vendors plugins integration.
 //* CLI support.
 //* ORM, DAO, CRUD.
 //* Install/update automatic.
 * Divers tools.
